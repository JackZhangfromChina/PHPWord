欢迎来到 PHPWord 文档
==================================

|PHPWord|

PHPWord是一个用纯PHP编写的库，它提供了一组类来写入和读取不同的文档文件格式。当前版本
PHPWord支持Microsoft Office Open XML (OOXML或OpenXML)，OASIS Open
Office应用程序 (OpenDocument或ODF) 的文档格式和富文本格式 (RTF)。
